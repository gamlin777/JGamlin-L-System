To run the program sucessfully please:

1) Install the DirectX 9 (August 2009) SDK
2) Add the 'Include' and 'x86' folders to your project directory
  i) Open Visual C++ 2010 and the project
  ii) Right click the project 'rvtsD3dApp' on the solution explorer and click 'Properties'
  iii) Go into: 'Configuration Properties then 'VC++ Directories'
  iv) Add the path to the 'Include' folder to 'Include Directories'
  v) Add the path to the 'x86' folder to the 'Library Directories'
3) Make sure you have the 'Config' Folder with the six 'exampleNum.txt' files
